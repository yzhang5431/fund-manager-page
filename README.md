# 📊 基金交易记录管理系统

一个简洁易用的基金交易记录管理工具，支持交易记录、持仓管理、收益分析和云端同步。

![Version](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 主要功能

### 💰 交易记录管理
- ✅ 记录买入/卖出交易
- ✅ 自动计算成本、收益和收益率
- ✅ 编辑和删除交易记录
- ✅ 基金代码自动查询名称
- ✅ 高级筛选（日期范围、基金名称、交易类型）
- ✅ 表格排序功能

### 📈 持仓管理
- ✅ 管理当前持有的基金
- ✅ 自动计算浮动盈亏
- ✅ 一键同步最新净值
- ✅ 持仓统计分析

### 📊 收益分析
- ✅ 基金收益排行榜
- ✅ 交易次数统计
- ✅ 最高/最低收益记录

### ☁️ 数据管理
- ✅ 本地自动保存
- ✅ 导出/导入 JSON 数据
- ✅ WebDAV 云同步（支持坚果云）
- ✅ 多设备数据同步

### 📱 响应式设计
- ✅ 完美适配手机、平板、电脑
- ✅ 移动端优化界面
- ✅ 支持添加到主屏幕

## 📖 使用说明

### 基本操作

**添加交易记录**
1. 选择交易类型（买入/卖出）
2. 填写基金信息和交易数据
3. 点击"计算收益"预览结果
4. 点击"保存记录"

**管理持仓**
1. 切换到"持仓管理"标签
2. 添加持有的基金信息
3. 点击 🔄 同步最新净值
4. 查看实时浮动盈亏

**查看分析**
1. 切换到"收益分析"标签
2. 查看各基金收益排行
3. 了解交易表现

### 高级功能

**基金代码查询**
- 输入基金代码后点击 🔍 按钮
- 自动从天天基金获取基金名称

**WebDAV 云同步**
1. 点击右上角 ⚙️ 设置
2. 配置 WebDAV 服务器信息
3. 点击 ☁️ 云同步上传/下载数据

**数据筛选和排序**
- 使用搜索框快速查找
- 选择日期范围筛选
- 点击表头排序

## 🐛 常见问题

**Q: 数据会丢失吗？**  
A: 数据保存在浏览器本地，除非清除浏览器数据。建议定期备份。

**Q: 如何在多设备间同步？**  
A: 配置 WebDAV 云同步，或使用导出/导入功能。

**Q: WebDAV 连接失败 "Failed to fetch"？**  
A: 本地文件(file://)由于浏览器CORS限制无法使用WebDAV。
- **推荐方案**：部署到 GitHub Pages
- **临时方案**：使用本地服务器（双击"启动本地服务器.bat"）
- 详见《WebDAV问题解决指南.md》

**Q: 交易记录和持仓如何关联？**  
A: 买入自动增加持仓份额，卖出自动减少持仓份额。系统通过基金代码或名称匹配。

**Q: 基金代码查询失败？**  
A: 检查网络连接，确认基金代码正确（6位数字）。

**Q: 可以多人使用吗？**  
A: 当前是单用户系统，每个人的数据独立存储。

更多问题请查看 [使用说明.txt](使用说明.txt) 或 [WebDAV问题解决指南.md](WebDAV问题解决指南.md)
